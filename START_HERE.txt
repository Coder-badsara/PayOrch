                      HOW TO RUN THE SYSTEM
═══════════════════════════════════════════════════════════════════════════════

OPTION 1: ONE-CLICK STARTUP (Easiest - Windows)
────────────────────────────────────────────────
From project root directory, double-click or run:
  start-servers.bat

This will:
  1. Open Backend server window (port 8000)
  2. Open Frontend server window (port 5173 or 5174)
  3. Both display startup messages
  4. System is immediately ready to use

OPTION 2: MANUAL STARTUP (Any Operating System)
────────────────────────────────────────────────
Terminal 1 - Start Backend:
  $ cd backend
  $ python manage.py runserver 0.0.0.0:8000

  Expected output:
  Starting development server at http://0.0.0.0:8000/
  Quit the server with CTRL-BREAK.

Terminal 2 - Start Frontend:
  $ cd frontend
  $ npm run dev

  Expected output:
  ➜ Local:   http://localhost:5173/
  press h + enter to show help
